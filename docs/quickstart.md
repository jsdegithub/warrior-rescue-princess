# 快速开始指南

## 🚀 快速启动

### 准备工作
确保你已安装以下工具之一：
- [HBuilderX](https://www.dcloud.io/hbuilderx.html) （推荐，可视化IDE）
- 或 Node.js + npm（命令行方式）

## 方式一：HBuilderX（最简单）

1. **打开项目**
   - 下载并安装 HBuilderX
   - 将 `warrior-rescue-princess` 文件夹拖入 HBuilderX

2. **运行到浏览器**
   - 点击工具栏的"运行" -> "运行到浏览器" -> 选择浏览器
   - 游戏将自动在浏览器中打开

3. **运行到微信小程序**
   - 点击"运行" -> "运行到小程序模拟器" -> "微信开发者工具"
   - 首次需要配置微信开发者工具路径

## 方式二：命令行（开发者）

1. **安装依赖**
```bash
cd warrior-rescue-princess
npm install
```

2. **运行开发服务器**
```bash
# H5版本
npm run dev:h5

# 微信小程序版本
npm run dev:mp-weixin
```

3. **构建生产版本**
```bash
# H5版本
npm run build:h5

# 微信小程序版本
npm run build:mp-weixin
```

## 🎮 游戏说明

### 目标
控制勇士从左侧出发，穿越重重关卡，到达右侧与公主相拥即可通关！

### 操作方式

**PC端：**
- `← →` 或 `A D`：左右移动
- `空格`：跳跃
- `J`：攻击

**移动端：**
- 使用屏幕下方的虚拟按钮控制

### 游戏元素
- ❤️ **生命值**：3颗心，触碰敌人或尖刺会失去1颗心
- 🏔️ **平台**：需要精准跳跃到达
- ⚠️ **尖刺**：触碰会受伤
- 🕳️ **深坑**：掉落即失败
- 👹 **敌人**：可以跳跃踩踏或攻击击败

## 📱 支持的平台

## 🔧 常见问题

### Q: 游戏画面显示不正常？
A: 确保浏览器支持Canvas API，推荐使用Chrome或Edge浏览器。

### Q: 移动端虚拟按钮不响应？
A: 检查是否正确配置了touch事件，在H5端测试时使用浏览器的移动设备模拟模式。

### Q: 小程序版本运行报错？
A: 不同小程序平台的Canvas API可能有差异，需要根据具体平台进行适配调试。

### Q: 图片资源加载失败？
A: 确保所有图片路径使用 `/static/assets/` 开头的绝对路径。
